# Model Generator

A tool for generating TypeScript models and interfaces.

## Setup

```bash
npm install
```

## Development

```bash
# Watch mode
npm run dev

# Build
npm run build

# Run tests
npm test

# Generate models
npm run generate
```

## Project Structure

- `src/` - Source code
- `tests/` - Test files
- `dist/` - Compiled output
- `coverage/` - Test coverage reports
