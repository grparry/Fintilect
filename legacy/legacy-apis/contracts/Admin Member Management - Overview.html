How to determine membership user id

The admin teller's login record for the old admin tool is stored in the table udt_AdminTable. Get the Teller field from that table for the teller logged in to the low code admin tool. The membership user id for that teller is the id column of the MembershipUsers table for the entry where the LEGACY_udt_AdminTable_ID is set to the Teller field from the udt_AdminTable.

Example using SQL

select MembershipUserId from MembershipUsers
where LEGACY_udt_AdminTable_ID = (select Teller from udt_AdminTable where EmailAddress = '{replaceMe}')

Member Search

Search for a member by an identifier. The identifier type is optional. Teller id for this and other calls refers to the MembershipUserId for the teller performing the search.

Route

POST ~/api/admin-member-management/v1/search

Request

Body
{
    "Identifier": long,  REQUIRED
    "IdentifierType": AdminMemberSearchIdentifierType,
    "TellerId": long  REQUIRED
}

AdminMemberSearchIdentifierType Possible values
    AccountNumber
    PersonNumber
    OrgNumber
    MemberNumber
    CoreUserId
    Username
    TaxId
    Uuid

Response
200 - OK - Success

Body
{
    "members": [ (this is a list)
        {
            "accountNumber": string,
            "personNumber": string,
            "username": string,
            "uuid": string,
            "accountCategory": string
        }
    ],
    "demographics": { (only returned when there is exactly 1 member in the response)
        "firstName": string,
        "middleInitial": string,
        "lastName": string,
        "address": {
            "category": string,
            "addressLine1": string,
            "city": string,
            "state": {
                "name": string,
                "abbreviation": string
            },
            "zipCode": {
                "five": string,
                "plusFour": string
            },
            "isExpired": bool,
            "isDefault": bool,
            "residenceYears": int
        },
        "phoneNumber": {
            "type": string,
            "isDefault": bool,
            "areaCode": string,
            "exchangeCode": string,
            "subscriberNumber": string,
            "number": string
        },
        "dateOfBirth": datetime,
        "emailAddress": {
            "type": string,
            "isDefault": bool,
            "address": string
        },
        "preferredContactMethod": string,
        "onlineBankingEnrollmentDate": datetime,
        "taxId": string
    }
}

401 - Unauthorized - The necessary configs for Member Management API are not enabled
500 - Internal Server Error - Something went wrong

Reset Password

Resets a password for the specified member to a provided password or a generated one if not provided. There is also the option to send the password to the member's device (SMS, phone, or Email) if the device Id is provided.

Route

POST ~/api/admin-member-management/v1/reset-password

Request

Body
{
    "Uuid" : long,   REQUIRED
    "AccountNumber" : int,   REQUIRED
    "Password" : "string",
    "DeliveryAddress" : "string", (Phone Format: "1231231234 or Email Format "test@email.com")
    "SendAsVoice" : bool,
    "ClientIp" : "string",
    "WebId": "string",
    "TellerId" : long   REQUIRED
}

Response
200 - OK - Success

Body
{
    "passwordResetSuccessful": bool,
    "messageDeliverySuccessful": bool
}

401 - Unauthorized - The necessary configs for Member Management API are not enabled
400 - Bad Request - Both the account and UUID were not provided
500 - Internal Server Error - Something went wrong
